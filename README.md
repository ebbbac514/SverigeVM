## 📖 Om sidan

Den 31 mars 2026 besegrade Sverige Polen med **3–2** på Strawberry Arena och säkrade en plats i fotbolls-VM 2026 i USA, Kanada och Mexiko. Viktor Gyökeres avgjorde i den 88:e minuten.

Den här sidan är en hyllning till:
- 🏆 **Matchen** – play-off finalen mot Polen, mål för mål
- 👑 **Graham Potter** – förbundskaptenen som vände allt
- ⚽ **Viktor Gyökeres** – hjälten som bar upp ett helt folk
- 📜 **Historiska minnen** – från 1958 till 2026
- 🌎 **VM 2026** – vad väntar i Grupp F

## 🛠 Teknisk info

Ren HTML5 + CSS3 + Vanilla JavaScript. Inga ramverk, inga beroenden (förutom Google Fonts).

```
sverige-vm2026/
├── index.html        ← Huvud-HTML
├── css/
│   └── style.css     ← All styling
├── js/
│   └── main.js       ← Animationer, konfetti, scroll-reveals
└── README.md
```

### Features
- **Konfetti-animation** vid sidladdning (blågult + guld)
- **Scroll-triggered reveals** för alla sektioner
- **Parallax mouse-tracking** på hero-bakgrund
- **Animated counters** på statistik-siffror
- **Fullt responsiv** – mobilanpassad design
- **Dark editorial design** med Playfair Display + EB Garamond

## 🚀 Kör lokalt

```bash
# Klona repot
git clone https://github.com/DITT-KONTO/sverige-vm2026.git

# Öppna i webbläsaren
open index.html
# – eller –
npx serve .
```

### Deploy på GitHub Pages
1. Gå till Settings → Pages
2. Välj `main`-branchen och `/` (root)
3. Spara – klart!
